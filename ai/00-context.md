# Project Context for AI

This is a personal-use project.

Primary goals:
- Build calm, distraction-free software
- Optimize for clarity and maintainability
- Avoid overengineering
- Prefer boring, readable solutions

Constraints:
- Single user
- No analytics
- No monetization
- No scale concerns
- No AI features by default

Assume:
- Code will be read more than written
- Simplicity is more valuable than cleverness
